# Arduino-MIDI-Controller
Arduino MIDI Controller

# Materials:
```
1. Jumper Cables
2. 10k Resistors
3. 300 Resistors
4. 5 Pin
```

# Instructions:
```
1. Open MIDI-Controller.ino in Arduino IDE
2. Verify then Upload to Arduino UNO
3. Connect MIDI Cable to MIDI INPUT on Audio Interface, Sound Module, Synthesiser, etc.
```
